# AI 智能写作工具站 — 设计文档

## 概述
零成本部署的 AI 写作工具站，提供周报生成、小红书文案、朋友圈文案三个工具。前 10 次免费，后续按次/包月收费。

## 技术栈
- **前端：** 纯静态 HTML + CSS + JS（无框架）
- **后端 API：** Cloudflare Workers（免费，10 万次/天）
- **AI 模型：** DeepSeek API（免费无限）
- **部署：** GitHub Pages（前端托管）+ Cloudflare Workers（API 代理）
- **存储：** localStorage（使用计数），后期可接 KV

## 架构

```
用户浏览器 → GitHub Pages (静态页面)
                ↓
         Cloudflare Workers (API 代理 + 限流)
                ↓
           DeepSeek API (AI 生成)
```

## 工具列表
1. **周报生成器** — 输入工作内容 → 输出结构化周报
2. **小红书文案** — 输入主题 → 输出小红书风格文案 + 标题 + 标签
3. **朋友圈文案** — 输入想法 → 输出朋友圈文案 + 配图建议

## 收费模式
- 新用户：10 次免费体验
- 按次：1 元/次（微信收款码手动确认后手动增加次数）
- 包月：19.9 元/月（后期实现）

## 页面结构
- `/` — 首页，三个工具卡片，显示剩余次数
- `/tools/weekly` — 周报生成器
- `/tools/xiaohongshu` — 小红书文案
- `/tools/moments` — 朋友圈文案
- `/pricing` — 价格页 + 微信收款码
